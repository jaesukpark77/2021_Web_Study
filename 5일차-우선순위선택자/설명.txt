- div 태그
: 분할 태그, 상자태그

div태그를 사용하기 위한 3가지 css 속성
width : 가로값
height : 세로값
background-color : 배경색


- span 태그
: 단순글자태그
혼자서 사용하면 아무런 의미가 없는 태그이지만 다른 태그 안에 들어가서 그부분만 내가 원하는 css속성을 적용하고 싶을 때 사용하는 태그

color / css 속성의 경우 태그에 사용하면 태그안에 있는 모든 글자가 전부 동일한 색으로 변경이 되는데 span 태그를 사용하면 그 부분만 다른색으로 변경하거나 기타 여러가지 css속성을 적용할 수 있다.

★ background-color
: 선택한 html 태그에 배경색을 넣어주는 css 속성

-우선순위선택자
: 선택자에 따라서 우선적으로 속성이 적용되는 순서

css속성은 동일한 대상한테 동일한 css속성을 넣어주면 등급이 같은 기준에서 무조건 밑에 있는 css 속성이 적용된다.

우선순위선택자 등급표

1등 #아이디
2등 .클래스
3등 태그명

- 선택자의 종류
1. #(아이디)
:나만 갖을 수 있으며, 하나만 사용가능. 고유명사
  적용방법 - <태그명 id="아이디명">
  사용방법 - #아이디명{속성:속성값;} ex) #div
{속성:속성값}-아이디 div
  읽는방법 - 아이디 아이디명
(한글 아이디 쓰지마세요)

★주의사항 : 똑같은 이름은 사용하지 말 것

2. 클래스
: 여러개 사용가능하며, 공유가능.
  적용방법 - <태그명 class="클래스명">
  사용방법- .클래스명{속성:속성값;} ex) .p{속성:속성값;} - 클래스 p
  읽는방법 - 클래스 클래스명

아이디 = 동일한 이름 못쓴다!!!
-> 웹 사이트에 존재하는 특별한 컨텐츠

클래스 = 동일한 이름을 사용해야지 의미 있음!!

쇼핑물에 상품 카테고리

아이디와 클래스는 서로 다른 존재이기 때뭉네 이름이 같아도 상관 x
하지만 헷갈릴 수 있으니 다른 이름 사용하자

★ 클래스는 한 칸만 띄워주면 동시에 여러개를 가질 수 있다.
