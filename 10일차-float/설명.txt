ul li?
목록 태그

ul 목록을 열러주는 명령어 같은 태그
li 목록안에서 실질적으로 내용물들을 담아주는 태그

목록태그도 display:block이기 때문에 div처럼 쓰면된다.
가로세로값 / 배경색 넣어주고 / 아이디 클래스 넣어줄 수 있음.

목록 태그는 그냥 사용하면 글자 왼쪽 동그라미 아이콘이 생긴다.
그래서 이것을  없애주는 css속성을 맨 처음에 사용한다!

*{margin:0;padding:0}
li{list-style:none}
a{text-decoration:none;color:black}
-> css 초기화 속성(각자의 태그가 가지고 있는 고유의 속성을 없애주는 구문)
맨 처음에 작성하고 시작해야한다!!


아무때나 사용하는게 아닌 목록이라고 부를 수 있는 존재들에만 사용한다
(목록 기준? 비슷한 내용을 가진 개체가 3개 이상 있으면 목록)

웹 사이트상에서 목록으로 지정된 내용들을 담아줄때 서용하는 태그

h - 제목태그 / 웹사이트 상에서 제목에 해당하는 존재들을 만들때 사용하는 태그

p - 문단 / 단락태그 / 웹사이트 상에서 문단은 나눠줄때 단락을 나눠줄때 사용한다.

a - 하이퍼링크태그 / 안에 적힌 내용을 클릭하면 href 경로 안에 작성한 곳으로 이동시켜주는 태그

span - 단순글자태그 / 혼자서 사용하지 않고 다른 태그에 들어가서 속성을 바꿔줄 때 사용하는 태그

div - 레이아웃 태그 / 상자태그 / 분할태그 / 웹사이트 상에서 큼지막한 구역을 나눠줄 때 사용하는 태그

img - 이미지태그 / 이미지를 출력해줌

ul li - 목록태그 / div태그가 공간을 분할해주면 그 안에서 실질적으로 내용물을 담아주는 그릇과 같은 역할 태그

웹사이트를 접속할 때 2가지 방법
1. html파일 클릭해서

2. 주소창에 주소를 입력해서 들어가는 방법
