h태그 = 글씨를 굵어지게 만드는건 같지만 제목에만 사용하는 태그
(display:block)

strong 태그 = 글자를 굵어지게 만드는 태그
주의사항 / 강조해야할 내용 등에 사용하는 태그
(display:inline)

b태그 = 글자를 굵어자개 만드는 태그
그냥 굵게 만들어주고 싶을때
(display:inline)

i태그 = 글자를 기울이는 태그
기술 용어 외국어 기타 등등 웹사이트 내에서 정보를 표시할 때 사용하는 태그

em태그 = 글자를 기울이는 태그
글을쓴 사람의 생각을 텍스트에 표현해서 강조할 때

ps.font-style:italic을 활용하여 기울여줄 수 있다

q태그 = 글자의 양쪽에 ""가 생기는 태그
인용문을 가져와 사용할 때 

blockquote태그 = 글자의 왼쪽에 긴여백이 들어가는 태그
인용문을 가져와 사용할 때 

mark태그 = 글자에 배경생이 들어가는 태그
강조하고 싶은 부분을 표시할 때
(background-color쓰면 더 편리하다)

cite태그 = 글자가 옆으로 기울어지는 태그
웹사이트 내에서 여러가지 정보에 대한 출처를 표시할 때 사용하는 태그

sub태그 = 글자가 굉장히 작아지며 오른쪽하단부로 위치하는 태그
어떠한 내용에 대한 짤막한 정보를 추가적으로 제공할 때 사용

s태그 = 글자에 중앙에 선을 그어주는 태그
취소선을 그어주는 역할
기존의 어떤 정보를 새로 갱신하면서 취소를 하는 역할
(물건판매 사이트에서 할인행사 상품 가격표기 부분)

ps.text-decoration:line-through를 사용해 똑같이 글자에 중앙에 선을 그어줄 수 있다

ins태그 = 밑줄이 생기는 태그
밑줄이 생기면서 강조를 해야하는 부분을 만들때 사용

del태그 = 글자의 중앙에 선을 그어주는 태그
취소선도 동일

ps. 물론 용도에 맞게 태그를 사용하는것도 중요하지만 실제로는  font-weight:bold 사용해서 글자를 굵게 만들어주는게 더 편리하다
